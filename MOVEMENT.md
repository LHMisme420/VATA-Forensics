# VATA Forensics Movement

We’re restoring a simple rule: **claims should be verifiable**.

## Principles
- Proof over authority
- Calm, reproducible methods
- Open verification for anyone
- No gatekeepers

## The Standard
A VATA artifact includes:
- the artifact (dataset/evidence bundle)
- deterministic hash (SHA-256)
- on-chain anchor tx
- verification steps anyone can run

## Call to Action
If you publish a claim, publish a VATA proof:
- hash
- tx
- verifier command
- (optional) evidence bundle

## Outcome
A public commons where evidence can’t be quietly rewritten.
